Ignite: Execute Java Methods Instantly with Arthas Integration

Ignite is a powerful developer tool designed for IntelliJ IDEA. It acts as a bridge between your code editor and the running JVM, allowing you to execute any Java method (Spring Beans, Services, Utils) directly from the IDE context menu without writing temporary controllers or test cases.

Under the hood, Ignite automatically leverages Arthas to attach to your process and invoke methods via vmtool.

Key Features

  • One-Click Execution: Right-click on any method name -> "Ignite Run". No more "Restart Server" just to test a small logic change.
  • Microservices Ready: Automatically detects all running Java processes. Select target service (PID) via a dropdown menu.
  • Smart JSON Editor: Built-in JSON editor with syntax highlighting and auto-formatting for method arguments.
  • Professional Console: Dedicated "Ignite Console" tool window to view structured execution reports (Parameters & Return Values formatted as JSON).
  • Parameter History: Remembers your last used arguments for every method.
  • Zero Configuration: Auto-downloads and attaches Arthas in the background. Handles Chinese character encoding automatically.
